# Fine-tuning PrithviT Models with TerraTorch

This guide provides a sample workflow for fine-tuning **PrithviT** models using **TerraTorch**.

---

## 1. Installation

### 1.1 Install Python
Make sure you have **Python 3.10+** installed on your system.  
You can verify your version with:
```bash
python --version
```

### 1.2 Install `uv`
`uv` is a fast Python package manager.

```bash
pip install uv
```

### 1.3 Sync the Environment
Once `uv` is installed, synchronize your project dependencies:
```bash
uv sync
```

This will create or update your virtual environment based on the project configuration (e.g., `pyproject.toml`).
